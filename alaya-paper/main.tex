\begin{abstract}
We present \alaya{}, an embeddable memory engine for conversational AI
agents that models memory as a dynamic process rather than a static store.
Inspired by Yogac\=ara Buddhist psychology's theory of storehouse
consciousness (\emph{\=alaya-vij\~n\=ana}) and grounded in computational
neuroscience, \alaya{} implements a three-store architecture---episodic,
semantic, and implicit---connected by a Hebbian graph overlay that reshapes
through use. The system provides five lifecycle operations beyond basic
storage and retrieval: consolidation (complementary learning systems
replay), transformation (deduplication, contradiction resolution, category
discovery), forgetting (Bjork dual-strength decay with retrieval-induced
suppression), preference emergence (gradual crystallization from accumulated
impressions), and emergent ontology (unsupervised category formation from
semantic node clustering). \alaya{} is implemented as a zero-dependency Rust
library with SQLite WAL and FTS5, requiring no external databases, no
network services, and optionally no LLM for memory management operations.
We evaluate baseline retrieval on LoCoMo and LongMemEval, demonstrating a
statistically significant retrieval crossover between full-context injection
and na\"ive RAG that motivates lifecycle-aware memory management. \alaya{}
is available as an open-source crate at
\url{https://crates.io/crates/alaya}.
\end{abstract}
