# Alaya v0.1.0 Publication Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Fix the co-retrieval bug, implement emergent ontology MVP (flat categories), add CI pipeline, and publish v0.1.0 to crates.io.

**Architecture:** Categories are flat (no hierarchy), stored in a new `categories` table with graph integration via `NodeRef::Category` and `LinkType::MemberOf`. Consolidation assigns new semantic nodes to existing categories; transform discovers new categories from uncategorized nodes via dual-signal clustering (embedding similarity + graph overlap). Graceful degradation: no embeddings falls back to graph-only, no LLM uses placeholder labels.

**Tech Stack:** Rust (stable), rusqlite 0.32, serde, thiserror 2.x, SQLite WAL + FTS5

**Design doc:** `docs/plans/2026-03-02-emergent-ontology-design.md`

```rust
#[test]
fn test_emergent_category_lifecycle() {
    let store = AlayaStore::open_in_memory().unwrap();

    // Store episodes with embeddings in a cluster
    for i in 0..5 {
        store.store_episode(&NewEpisode {
            content: format!("I made pasta with {} sauce", ["tomato", "pesto", "alfredo", "marinara", "bolognese"][i]),
            role: Role::User,
            session_id: "s1".to_string(),
            timestamp: 1000 + (i as i64) * 100,
            context: EpisodeContext::default(),
            embedding: Some(vec![0.9 + (i as f32) * 0.01, 0.1, 0.0]),
        }).unwrap();
    }

    // Consolidate with provider that returns semantic nodes with embeddings
    let provider = MockProvider::with_knowledge(vec![
        NewSemanticNode { content: "User cooks pasta".to_string(), node_type: SemanticType::Fact, confidence: 0.9, source_episodes: vec![EpisodeId(1), EpisodeId(2)], embedding: Some(vec![0.92, 0.08, 0.0]) },
        NewSemanticNode { content: "User likes Italian food".to_string(), node_type: SemanticType::Fact, confidence: 0.85, source_episodes: vec![EpisodeId(3), EpisodeId(4)], embedding: Some(vec![0.91, 0.09, 0.0]) },
        NewSemanticNode { content: "User knows many sauces".to_string(), node_type: SemanticType::Concept, confidence: 0.8, source_episodes: vec![EpisodeId(1), EpisodeId(5)], embedding: Some(vec![0.93, 0.07, 0.0]) },
        NewSemanticNode { content: "User enjoys cooking".to_string(), node_type: SemanticType::Fact, confidence: 0.75, source_episodes: vec![EpisodeId(2), EpisodeId(3)], embedding: Some(vec![0.89, 0.11, 0.0]) },
    ]);
    store.consolidate(&provider).unwrap();

    // No categories yet — consolidation doesn't create them
    assert!(store.categories(None).unwrap().is_empty());

    // Transform should discover a category from the clustered nodes
    let report = store.transform().unwrap();
    assert!(report.categories_discovered >= 1, "should discover cooking category");

    let cats = store.categories(None).unwrap();
    assert!(!cats.is_empty(), "should have at least one category after transform");

    // Second consolidation with more nodes: should assign to existing category
    for i in 5..10 {
        store.store_episode(&NewEpisode {
            content: format!("cooking episode {}", i),
            role: Role::User,
            session_id: "s2".to_string(),
            timestamp: 2000 + (i as i64) * 100,
            context: EpisodeContext::default(),
            embedding: Some(vec![0.91, 0.09, 0.0]),
        }).unwrap();
    }
    let provider2 = MockProvider::with_knowledge(vec![
        NewSemanticNode {
            content: "User experiments with recipes".to_string(),
            node_type: SemanticType::Fact,
            confidence: 0.85,
            source_episodes: vec![EpisodeId(6), EpisodeId(7)],
            embedding: Some(vec![0.90, 0.10, 0.0]),
        },
    ]);
    let report2 = store.consolidate(&provider2).unwrap();
    assert_eq!(report2.categories_assigned, 1, "new node should be assigned to existing cooking category");
}
```

### Task 11: CI Pipeline

**Files:**
- Create: `.github/workflows/ci.yml`

**Step 1: Write CI workflow**

```yaml
name: CI

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

env:
  CARGO_TERM_COLOR: always
  RUSTFLAGS: "-Dwarnings"

jobs:
  test:
    strategy:
      matrix:
        os: [ubuntu-latest, macos-latest, windows-latest]
        rust: [stable, "1.75.0"]
    runs-on: ${{ matrix.os }}
    steps:
      - uses: actions/checkout@v4
      - uses: dtolnay/rust-toolchain@master
        with:
          toolchain: ${{ matrix.rust }}
          components: clippy, rustfmt
      - uses: Swatinem/rust-cache@v2
      - name: Check formatting
        if: matrix.rust == 'stable' && matrix.os == 'ubuntu-latest'
        run: cargo fmt -- --check
      - name: Clippy
        if: matrix.rust == 'stable'
        run: cargo clippy --all-targets -- -W clippy::pedantic
      - name: Test
        run: cargo test --verbose
      - name: Doc tests
        if: matrix.rust == 'stable' && matrix.os == 'ubuntu-latest'
        run: cargo test --doc
      - name: No network deps
        if: matrix.rust == 'stable' && matrix.os == 'ubuntu-latest'
        run: |
          if cargo tree --no-default-features 2>/dev/null | grep -qE '(reqwest|hyper|h2)'; then
            echo "ERROR: Network dependencies found"
            exit 1
          fi
```
