\begin{abstract}
Large language model (LLM) agents increasingly require persistent memory
that goes beyond context-window chat history. Over 90 memory systems have
emerged since 2023, ranging from flat-file approaches (MEMORY.md) through
knowledge-graph engines (Zep/Graphiti, HippoRAG) to cognitively inspired
architectures (SYNAPSE, Alaya). Yet no unified taxonomy captures the full
landscape. We survey these systems along four axes drawn from cognitive
science and information retrieval: (1) memory \emph{structure} (episodic,
semantic, procedural, implicit), mapped to the CoALA framework; (2)
\emph{retrieval} architecture (sparse, dense, graph, fusion), mapped to
the RAG taxonomy; (3) memory \emph{lifecycle} (formation, consolidation,
forgetting, transformation); and (4) \emph{preference emergence} (how
behavioral patterns crystallize from interaction). We ground this taxonomy
in neuroscience (Hebbian learning, complementary learning systems, Bjork
dual-strength forgetting) and Yogac\=ara Buddhist psychology
(\emph{alaya-vij\~n\=ana}, \emph{v\=asan\=a}, \emph{b\=ija}), showing
that both traditions converge on the insight that memory is a dynamic
process, not a static store. We identify five convergent design principles
emerging independently in the practitioner community, catalog critical
gaps (principled forgetting, preference emergence, adaptive retrieval),
and propose a research agenda for the next generation of agent memory
systems.
\end{abstract}
