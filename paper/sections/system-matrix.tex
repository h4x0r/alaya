Three patterns emerge from the per-system view that are not visible in the
aggregate statistics.  First, contradiction resolution is the rarest
capability: only 4 of 41~systems implement it (Alaya, Mem0, Zep/Graphiti,
and Hindsight), all dedicated memory systems.  Second, Modular RAG systems
average 5.0/8 on the capability score, versus 4.0 for Advanced RAG and 1.9
for Naive RAG---retrieval sophistication correlates strongly with
architectural investment in lifecycle operations.  Third, no framework memory
module exceeds a score of~5: memory is a secondary concern in orchestration
frameworks, limiting investment in consolidation, contradiction resolution,
and graph-based retrieval.
