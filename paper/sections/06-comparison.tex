\subsection{Dominant Paradigms}

Five architectural paradigms have emerged:
\begin{enumerate}[leftmargin=*, nosep]
    \item \textbf{Vector store} (flat semantic retrieval)
    \item \textbf{Knowledge graph} (structured relationships)
    \item \textbf{OS-inspired tiering} (RAM/disk metaphor)
    \item \textbf{RL-trained memory policies} (learned management)
    \item \textbf{Parametric memory} (embedded in model weights)
\end{enumerate}
